# Simple Todo App

The app allows users to manage a list of "Todos" through a clean, responsive interface. It focuses on CRUD (Create, Read, Update, Delete) operations and client-side form validation.

## Functionality

Task Creation: Users can add new tasks by clicking an "Add" button, which opens a modal containing a form for the task name and a due date.

Task Completion: Each task has a checkbox that allows users to mark it as completed, which typically triggers a visual change (like a strikethrough).

Task Deletion: Users can remove individual tasks from the list using a delete icon.

Form Validation: The project implements custom JavaScript validation to ensure that the task name is provided and the date is valid before the "Submit" button becomes active.

Modal Logic: It features interactive popups (modals) that can be closed by clicking a close button, clicking the overlay, or pressing the Escape key.

## Technology

HTML5 & CSS3: Used for the structural layout and styling, employing BEM (Block Element Modifier) methodology for organized and maintainable CSS.

JavaScript (ES6): The core logic is built using Object-Oriented Programming (OOP) principles, specifically utilizing Classes (such as Todo and FormValidator) to keep the code modular and reusable.

Vite/Build Tools: Modern front-end tooling is used for development and bundling.

GitHub Pages: The project is deployed and hosted via GitHub's static site hosting service.
